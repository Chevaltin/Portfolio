Base de données de Minéralogie

Moi May 23, 2022
2 Minutes

Projet, SQL, Méthode Meurise

Développement d’une Base de Donnée de fossiles et de pierres précieuses

Pourquoi débuter ce projet ?

Étant un collectionneur amateur de fossiles et de pierres précieuses, un moyen de garder facilement des informations sur chacun de mes spécimens ainsi que les numéroter est quelque chose de vital, tous les collectionneurs de ma connaissance ont un système de ce type : ils numérotent tous leurs spécimens de façon à savoir quand ils les ont eu, s’ils les ont acheté ou récupéré par eux-même, etc. L’idée de faire une base de donnée de ma collection m’est donc venu tout naturellement.

Brève description du projet

Ce projet consistera donc en la création d’une base de donnée de ma propre collection de pierres précieuses et de fossiles à l’aide de la méthode Meurise et de PostrgresSQL. Une application permettant l’utilisation de cette base de donnée sera peut-être créée dans le futur mais mon objectif ici est tout d’abord la création de cette base de données et d’y insérer les données de tous mes spécimens, tels que leurs noms, leur composition, leur origines, des photographies, etc.

Découpage du projet

Dans un premier temps, un dictionnaire de données de toutes les informations que je souhaite insérer dans ma base de données sera effectuer puis, à partir de ce dictionnaire de données et de contraintes que je définierai moi-même, je créerai le graphe des dépendances fonctionnelles, suivi du Modèle Conceptuel de Données de ma base de données. Ce MCD me permettra donc de créer un Modèle Logique de Données, sur lequel je me baserai, ainsi que sur les contraintes définies dans le dictionnaire des données, pour faire le script de création de la base de données.

Ensuite, je réunirai toutes les informations sur chaque pierre et chaque fossile que je possède pour ensuite les insérer dans la base de données avec un script.

Ce que le projet va m’apprendre

Ce projet va me permettre de mieux comprendre comment faire des bases de données à l’aide de la méthode Meurise, ayant moi-même quelques difficultés avec cette méthode. Il va aussi me permettre de revoir plus en détail la création d’une base de données et l’insertion de données dans la-dite base de données à l’aide de PostgresSQL.
